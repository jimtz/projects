Καλησπέρα
Έκανα μια προσπάθεια να λύσω την άσηση που μου στείλατε. 
Αρχικά, χρησιμοποιώντας java έφτιαξα τις απαραίτητες κλάσσεις. Χρησιμοποίησα
SpringBoot. Αρχικά προσπάθησα με Controller στο αρχείο home.jsp να δημιουργήσω 
διαδραστικότητα, αλλά δε τα κατάφερα, οπότε με τους Controllers στέλνω τα δεδομένα μέσω ενός Json αρχείου 
στο localhost:8080 και το αποτέλεσμα παράγεται χρησιμοποιώντας  react ,ένα framework της javascript.
στο localhost:3000. Δεν είμαι τόσο εξοικειωμένος με react ,αλλά έκανα μια προσπάθεια.
Ευχαριστώ για την ευκαιρία να συμμετέχω στην άσκηση.
Καλή συνέχεια!
